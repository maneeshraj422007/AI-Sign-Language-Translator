\# AI-Based Real-Time Sign Language Translator



An AI-powered application that recognizes sign language gestures in real time using Computer Vision and Deep Learning, translating them into text and speech.



\## Tech Stack



\- Python

\- OpenCV

\- MediaPipe

\- TensorFlow / Keras

\- Flask

\- HTML

\- CSS

\- JavaScript



\## Features



\- Real-time hand detection

\- Gesture recognition

\- Text generation

\- Speech output

\- Multilingual support
